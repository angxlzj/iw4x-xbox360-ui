# Modern Warfare 2 XBOX 360 User Interface
IW4x Xbox 360 UI Implementation by angxlzj

#

## Installation Guide
Implement the folder *userraw/ui_mp* inside the *userraw* folder on your *iw4x-rawfiles-client* installation location folder.
